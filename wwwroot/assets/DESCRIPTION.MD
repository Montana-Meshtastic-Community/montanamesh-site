# Website Asset Bundle

This folder contains public static assets for the MontanaMesh website.

## Contents

- Favicon and app icon files.
- Web manifest.
- Logo images.
- `fonts/` with Iosevka font files.
- `images/` with image assets used by the layout.
- `images/backgrounds/` with optimized local WebP page backgrounds.
- `js/` legacy or alternate JavaScript asset path.

## Maintenance Rule

Whenever images, icons, manifest data, fonts, or asset paths change in this folder, update this `DESCRIPTION.MD` in the same change.
