# Website CSS

This folder contains site-wide CSS for the MontanaMesh website.

## Current Role

`site.css` controls the visual theme, layout, local WebP page backgrounds, cards, homepage information-card spacing, navigation styling, responsive behavior, and light/dark presentation used by the shared layout and public pages.

## Maintenance Rule

Whenever styling, layout behavior, responsive rules, or theme behavior changes in this folder, update this `DESCRIPTION.MD` in the same change.
